# 3D Engine Base

Use MSYS2 on Windows

## Depedencies

- base-devel 
- mingw-w64-x86_64-toolchain
- mingw-w64-x86_64-cmake
- mingw-w64-x86_64-SDL2 or mingw-w64-x86_64-freeglut
- mingw-w64-x86_64-glew 
- mingw-w64-x86_64-mesa

## Compiling

For SDL2 version
```
make
```

For FreeGLUT version
```
make glut
```
